# 2024年秋季 - Linux基础课用笔记

## VM ware 虚拟机软件

### 概念

    允许用户在一台物理机上，安装、运行多个虚拟机(只要硬件资源足够多)，每个虚拟机都可以安装运行多个不同的操作系统，比如Linux、windows等，用到是些虚拟化技术，提高了硬件资源利用率，降低成本，未开发、测试和学习提供了新的方案。
